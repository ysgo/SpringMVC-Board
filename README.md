- Client

  - HTML5 / CSS3 / JavaScript / jquery
  - JSP
  
- Server

  - Spring MVC FrameWork
  - Tomcat 9
  
- Database

  - Oracle 11g XE
  
- Development Flatform

  - jdk1.8.0_211
  - eclipse-jee-2018-12-R-win32-x86_64
  - Spring STS 3.9.9
  - sqldeveloper
  - Chrome
